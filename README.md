# File Organization
- [ZArchiver](https://zdevs.ru/en/) - A (really good) file manager made by ZDev (keep in mind this is closed sourced tho)
- [Material files](https://f-droid.org/packages/me.zhanghai.android.files/) - Open source file manager that also lets u access the files of the official fnf port (yay!!)
- [Files](https://play.google.com/store/apps/details?id=com.marc.files) - This is just a shortcut to your files for those who couldn't access them for some reason 

# Sprite Making

## Actually Drawing
- [IbisPaint X (SUBSCRIPTION)](https://ibispaint.com/) - Free raster drawing app that allows you to draw 
- [Procreate (PAID)](https://procreate.com/) - a polished drawing app exclusive to apple devices (unfortunately), with a main focus on ipads
- [Toonsquid (PAID)](https://toonsquid.com/) - another animation app similar to procreate that's exclusive to apple devices (unfortunately 2)
- [Flipaclip (SUBSCRIPTION](https://flipaclip.com/) - Simple animation app that uses frame by frames and it's been used by lots of people,, (if you've ever seen an animation meme, they probably used this)
- [Clip Studio Paint (SUBSCRIPTION)](https://play.google.com/store/apps/details?id=jp.co.celsys.clipstudiopaint.googleplay) - The closest android will get to what procreate does (probably)
- [Alight Motion (SUBSCRIPTION)](https://alightmotion.com/) - Video editing program where suprisingly you can make tweened animations with

## Exporting
- [Tex Packer EX](https://kitzsh.github.io/tex-packer-ex/) - textures to spritesheets packer that supports lots of formats outside fnf (USE _STARLING_ FOR FNF SPRITES)
> alternatively there is actual forks that focus on fnf but some of them were made by weirdos iirc so nah
- [Spritesheet and XML Generator](https://uncertainprod.github.io/FNF-Spritesheet-XML-generator-Web/) - allows you to combine frames to generate an fnf spritesheets

# Song Making
- [FL Studio Mobile](https://www.image-line.com/fl-studio-mobile) - Mobile version of [fl studio](https://www.image-line.com/) that supports dwps and ogg files
- [Garageband](https://www.apple.com/ios/garageband/) - A really good alternative exclusive to apple devices
- [BandLab](https://www.bandlab.com/) - A more simpler alternative that supports both and also has a web version

# FNF Engines
- [Friday Night Funkin' (V-Slice)](https://play.google.com/store/apps/details?id=me.funkin.fnf) - The official mobile port of FNF
      > btw this has its own mod page on gb!! [check it here](https://gamebanana.com/mods/cats/38080)
- [Pocket Engine](https://gamebanana.com/mods/406483) - An FNF engine that mimics the knockoff FNF apps on the play store but you can actually mod it
- [Funkin' Android (the older one)](https://github.com/luckydog7/Funkin-android) - The original unofficial android port by luckydog7 that predates the official port
- [Funkin' Android (0.4.X)](https://github.com/FunkinDroidTeam/Funkin) - The original fanmade port by ZackDroid's team that jumpstarted development to the official one
- [FunkinIOS (OUTDATED)](https://github.com/hadobedo/FunkiniOS) - An attempt to port luckydog7's fnf port to ios
